# Service Photos

Each service has its own folder. Drop service-specific images here.

## Folder structure

```
services/
├── stick-framing/         ← Stick Built Framing (wood frame in progress)
├── prefab-framing/        ← Pre-Fab Framing (truss installation)
├── renovations/           ← Renovations (interior work)
├── decking-pergolas/      ← Decking & Pergolas
└── material-supply/       ← Material Supply (timber, hardware)
```

## How to upload

1. Click into the folder for the service (e.g. `stick-framing/`)
2. Click **Add file → Upload files**
3. Drag your photo in
4. Commit

Once uploaded, tell Claude the filename and which service it belongs to — the services page will be wired up to use it.

## Recommended specs

- **Format**: JPG
- **Width**: 1600–2400px
- **Aspect**: 4:3 horizontal works best for the services layout
- **File size**: under 500 KB per photo
