# Project Photos

Each project has its own folder. Drop images into the folder for that project.

## Folder structure

```
projects/
├── deck/                  ← Deck & Pergola
├── renovation/            ← Renovation
├── bondi-extension/       ← Bondi Extension
├── double-bay/            ← Double Bay New Build
├── neutral-bay/           ← Neutral Bay Prefab Frame
└── kirribilli-pergola/    ← Kirribilli Pergola
```

## How to upload

1. Click into the folder for the project (e.g. `hawthorne-deck/`)
2. Click **Add file → Upload files**
3. Drag your photo(s) in — name them clearly: `1.jpg`, `2.jpg`, `hero.jpg`, etc.
4. Commit

## Recommended specs

- **Format**: JPG (smaller) or PNG (transparent)
- **Width**: 1600–2400px for retina-ready display
- **Aspect**: 4:3 horizontal works best for the carousel
- **File size**: under 500 KB per photo (compress with tinypng.com if needed)

Once uploaded, tell Claude the filenames and which project they belong to — they'll be wired into the carousel and project cards.
